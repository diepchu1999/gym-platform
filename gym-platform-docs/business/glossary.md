# Glossary

## Member
A customer who has registered in the system. Member can be trial, active paid member, VIP, student, inactive, or suspended.

## Lead
A potential customer before becoming a registered member.

## CCCD KYC
Identity verification using Vietnamese citizen ID information. Required for trial package and can be required for contract/installment.

## Student Verification
Verification based on student card to qualify for student discount.

## Main Gym Package
Monthly, quarterly, yearly, or VIP package that grants gym access across branches.

## Trial Package
Free 7-day package requiring CCCD KYC. Allows 1 check-in per day and 1 group class trial session.

## VIP
Premium member tier with private room quota and 3 free massage bookings per week.

## Group Class
Scheduled class such as Yoga or Kickboxing. Sold by number of sessions through class pass.

## PT
Personal Trainer service. 1-on-1, 90 minutes per session, bookable from 06:00 to 22:00.

## Private Room
Limited gym room for private workout. Must be booked by hour. Max 2 hours per booking.

## Pantry
Food/drink/protein service inside gym. Available to all members from 06:00 to 22:00.

## Booking Engine
Shared module that handles slot availability, resource conflict, booking status, cancellation, no-show, quota, and payment integration.

## CSKH
Customer care team responsible for follow-up, no-show calls, complaints, tickets, and retention.

## Sale Branch
Branch where the package/order was sold.

## Home Branch
Primary branch associated with the member.

## Check-in Branch
Branch where the member actually enters and uses gym facilities.

## Quota
Limited included usage allowance, such as VIP private room hours/month or massage free sessions/week.

## No-show
Customer did not arrive for booking after CSKH confirmation window.
