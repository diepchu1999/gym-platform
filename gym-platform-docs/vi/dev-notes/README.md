# Dev Notes (Ghi chú phát triển)

> Bản tiếng Việt (canonical). English: [`../../en/dev-notes/README.md`](../../en/dev-notes/README.md).

Tạo một file markdown sau mỗi phiên phát triển quan trọng.

Định dạng:

```text
YYYY-MM-DD-module-summary.md
```

Bao gồm:
- Đã implement gì.
- File nào thay đổi.
- Migration database đã thêm.
- Test đã thêm.
- Business rule đã implement.
- Vấn đề đã biết.
- Bước tiếp theo.
